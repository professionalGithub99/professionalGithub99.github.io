Detailed Description

Singleton class containing all data structures necessary for unfolding. The core data structures include a response_matrix, light_spectrum,.

This class is used to aggregate various classes that are part of a neutron unfolding such as response matrix, light output spectrum, neutron guess spectrum and more

UNFOLD::CORE_SINGLETON::set_light_spectrum ( Eigen::MatrixXf  _response_matrix,
Eigen::VectorXf  _neutron_spectrum_ground_truth 
)
private

This function is automatically called when the ground_truth spectrum is set. It multiplies the ground truth spectrum by the response matrix and uses each bin in the new vector as a the mean for generating a poisson random variable for a new vector corresponding to the respective bin number. This new vector is the light_spectrum.

Parameters
[in]_response_matrixThe response matrix used to generate the light spectrum.
[in]_neutron_spectrum_ground_truthThe ground truth neutron spectrum.
