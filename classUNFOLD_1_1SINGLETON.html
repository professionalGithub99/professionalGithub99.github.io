Detailed Description

Singleton class containing all data structures necessary for unfolding. The core data structures include a response_matrix, light_spectrum,.

This class is used to aggregate various classes that are part of a neutron unfolding such as response matrix, light output spectrum, neutron guess spectrum and more

UNFOLD::SINGLETON::SINGLETON ( TString  _response_matrix_file)

Constructor for SINGLETON class.

Initialize the SINGLETON class with a text file containing the response matrix such that rows are delimited by spaces and columns are delimited by newlines. Row count corresponds to number of neutron bins while column count correspnod to number of light bins.
