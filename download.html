Download and Installation

Dependencies

Please make sure you have the following dependencies installed on your system before proceeding:

  • CMake: https://cmake.org/download/
  • Root Data Analysis Framework: https://root.cern/install/

The following dependencies are already included in the code and require no action for installation. We leave it in here for developers who may want to modify the code or understand the underlying implementation.

  • Eigen3: https://eigen.tuxfamily.org/index.php?title=Main_Page#Overview

Installation

After cloning the repository, please follow the following steps to install the package:

mkdir build
cd build
cmake ../
cmake --build . -jN

Note the N in -jN is the number of cores you want to run for ex. N=4
Add the following line to your rootlogon.C or run the command when you open a ROOT environment:

gSystem->Load("path/to/nunf/build/libmyUnfold.so");

For an example of unfolding experimental data please see running a basic unfolding example from expiremental data.
For an example of unfolding simulated data please see running a basic unfolding example using simulated data. For a tutorial on matrix condition please see matrix condition tutorial. For a tutorial on uncertainty quantification please see uncertainty quantification tutorial.
