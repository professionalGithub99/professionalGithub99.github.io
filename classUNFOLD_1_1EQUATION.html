Detailed Description

UNFOLD class containing all data structures necessary for unfolding. The core data structures include a response_matrix, light_spectrum,.

This class is used to aggregate various classes that are part of a neutron unfolding such as response matrix, light output spectrum, neutron guess spectrum and more

Constructor for UNFOLD class.

Initialize the UNFOLD class with two text files, one containing the response matrix such that rows are delimited by spaces and columns are delimited by newlines. Row count corresponds to number of neutron bins while column count correspnod to number of light bins. Another contains a light spectrum file used for unfolding.
